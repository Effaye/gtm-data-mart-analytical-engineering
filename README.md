# GTM Data Mart

This repository contains the analytic mart for sales channel efficiency, marketing strategy, and account prioritization.

## Structure
- `data/` — Raw and processed data files
- `scripts/` — ETL and transformation scripts
- `notebooks/` — Analysis and exploration notebooks
- `config/` — Configuration files (e.g., database, environment)
- `tests/` — Test scripts for validation
- `notes` — Project notes and design

## Main Features
- Channel funnel reporting
- Opportunity scoring
- Ideal customer profiling
- Sales and marketing efficiency analysis

## Getting Started
1. Place your raw data in `data/`
2. Use scripts in `scripts/` for ETL
3. Explore and analyze in `notebooks/`
4. Adjust settings in `config/`
5. Run tests in `tests/`

---
See `notes` for detailed design and metrics.
